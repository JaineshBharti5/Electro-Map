# ⚡ Electro-Map

Electro-Map is a web-based application that helps users locate and explore electric vehicle charging stations using an interactive map, route planning, and user management features.

---

## 🚀 Features

- 📍 Interactive map to find EV charging stations  
- 🔍 Search and filter stations  
- 🛣️ Route planning  
- 🔐 User authentication (Login/Signup)  
- 🛠️ Admin panel to manage stations  

---

## 🧑‍💻 Tech Stack

**Frontend:**
- React (Vite)
- Tailwind CSS

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB
